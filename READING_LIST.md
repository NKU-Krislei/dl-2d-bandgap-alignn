## ① A Gentle Introduction to Graph Neural Networks
**——理解"图"和"消息传递"的直觉**

| 项 | 内容 |
|----|------|
| **来源** | Google Brain 团队，交互式网页版 |
| **链接** | https://distill.pub/2021/gnn-intro/ |
| **类型** | 在线交互式教程（非传统论文） |
| **阅读时间** | 20–30 分钟 |

### 内容概要
用动画和交互式图表，从零解释：
- **图是什么**：节点（entity）+ 边（relation）
- **消息传递（Message Passing）**：每个节点收集邻居信息，更新自己的"状态向量"
- **图级读出（Readout）**：把所有节点的信息汇聚成一个整体表示
- **不同 GNN 变体**：GCN、GAT、GraphSAGE 等的直觉差异

### 读的目的
你不需要看任何深度学习教材。**这篇就够了。** 它用最直观的方式帮你建立 GNN 的"心理模型"：

> **物理类比**：想象晶格中的每个原子都有一种"电子云状态"。在自洽场迭代中，每个原子会"感知"周围邻居的势场，更新自己的波函数——**这和消息传递在数学上是对偶的**。

读完这篇后，你应该能回答：GNN 中的一层 message pass 在做什么？为什么要做多层？readout 怎么得到全局属性？

## ② SchNet: A continuous-filter convolutional neural network for modeling quantum interactions
**——理解"怎么把晶体坐标喂给神经网络"**

| 项 | 内容 |
|----|------|
| **作者** | K.T. Schütt et al. |
| **来源** | NeurIPS 2017 |
| **DOI** | 无（会议论文，直接搜标题即可下载） |
| **代码** | https://github.com/atomistic-machine-learning/schnetpack |
| **阅读时间** | 30–40 分钟（重点读 Section 2–3） |

### 内容概要
- **连续滤波卷积层**：不像传统 CNN 对固定网格做卷积，SchNet 对**连续的 3D 空间坐标**做卷积——天然适合原子坐标输入
- **截断半径内建图**：以每个原子为中心，在一定截断距离内的所有邻居原子构成其"感受野"
- **平移不变性**：通过只使用原子间**相对距离**（而非绝对坐标）作为输入特征，天然满足平移对称性

### 读的目的
这是从"通用 GNN"到"适合晶体的 GNN"的关键一步。你作为物理人会很欣赏它对对称性的处理：

> **物理直觉**：SchNet 的 continuous-filter convolution = 把截断半径内的邻居原子按距离加权求和。这和 DFT 中用赝势/PAW 方法处理近邻相互作用的方式在精神上是一致的。

**重点段落**：Equation (2)–(3)——这是整个 continuous-filter convolution 的核心公式，展示了 $h_i^{(t+1)}$ 如何由 $h_j^{(t)}$ 和 $d_{ij}$ 更新。

## ③ Neural Message Passing for Quantum Chemistry (MPNN 框架)
**——理解消息传递的数学框架（统一视角）**

| 项 | 内容 |
|----|------|
| **作者** | J. Gilmer, S. Schoenholz, P.F. Riley, O. Vinyals, G.E. Dahl |
| **来源** | ICML 2017 |
| **DOI** | 无（会议论文，arXiv:1704.01212） |
| **阅读时间** | 30–40 分钟（重点读 Section 2） |

### 内容概要
这篇论文的价值不在提出新模型，而在**统一**：
- 把当时所有的图上消息传递方法（GGNN、MPNN、Interaction Networks 等）**归纳成一个通用数学框架**
- 框架只有两个核心步骤：
  1. **消息传递**：$m_i^{(t+1)} = \sum_{j \in \mathcal{N}(i)} M_t(h_i^{(t)}, h_j^{(t)}, e_{ij})$
  2. **读出**：$\hat{y} = R(\{h_v^{(T)}\})$
- 然后展示：不同的具体方法只是 $M_t$（消息函数）和 $R$（读出函数）的不同选择

### 读的目的
**这篇是"字典"。** 读完之后你再去看任何 GNN 论文（CGCNN、ALIGNN、SchNet），都能立刻把它们归入这个框架，说出"哦，它的 message function 是 X，readout function 是 Y"。

> **与物理的对应**：
> | GNN 概念 | 物理类比 |
> |----------|---------|
> | 节点特征 $h_i^{(t)}$ | 原子轨道基组 |
> | 消息函数 $M_t$ | 哈密顿量矩阵元 $H_{ij}$ |
> | 消息传递 T 层 | 自洽场迭代 T 步 |
> | 读出函数 $R$ | 计算观测量 $\langle \psi | \hat{O} | \psi \rangle$ |

## ④ CGCNN: Crystal Graph Convolutional Neural Networks for an Accurate and Interpretable Prediction of Material Properties
**——理解"晶体结构 → 图"的具体编码方式（开山之作）**

| 项 | 内容 |
|----|------|
| **作者** | Tian Xie, Jeffrey C. Grossman (MIT) |
| **来源** | Physical Review Letters 120, 145301 (2018) |
| **DOI** | 10.1103/PhysRevLett.120.145301 |
| **代码** | https://github.com/txie-93/cgcnn |
| **阅读时间** | 45–60 分钟（全文，重点 Fig. 1 + Equation (1)–(4)） |

### 内容概要
**这是材料信息学中 GNN 的"第一性原理"论文。** 它第一次系统地回答了：
- **节点怎么编码**？每个原子用一组预定义的属性（原子序数、周期表行/列、电负性、价电子数等 9 个特征）作为初始向量
- **边怎么编码**？两个原子之间的**相对距离和方向**被离散化为高斯基函数展开（14 个基函数）
- **消息传递怎么算**？结合邻居节点特征 + 边特征（距离信息），通过可学习的权重矩阵更新节点特征
- **全局特征**：池化后加上**全局材料特征**（空间群号、原子种类数等）
- **读出预测**：全连接层 → 标量输出（形成能、带隙等）

### 性能基准（需要记住的数字）
| 属性 | CGCNN MAE | 传统指纹法 MAE |
|------|-----------|---------------|
| 形成能 | ~0.039 eV/atom | ~0.156 eV/atom |
| 带隙 | ~0.25–0.26 eV | ~0.39 eV |

### 读的目的
这篇论文的价值在于**可解释性**——你可以通过可视化学到的原子特征向量，看到网络"理解"了什么化学信息（如原子尺寸、电负性趋势）。论文的 **Fig. 3**（特征可视化）特别值得看。

> **对你说**：这篇论文写得很物理-friendly。Xie 是物理背景出身，论文语言和固体物理论文的叙述方式很像。读完它你就彻底理解了"晶体图表示"这个概念。

## ⑤ ALIGNN: Atomistic Line Graph Neural Network for improved materials property predictions
**——你的项目的核心模型（必读）**

| 项 | 内容 |
|----|------|
| **作者** | Kamal Choudhary, Brian DeCost, Chenming Huang, Brian W. Olson, etc. (NIST) |
| **来源** | npj Computational Materials 7, 185 (2021) |
| **DOI** | 10.1038/s41524-021-00650-1 |
| **代码** | https://github.com/usnistgov/alignn |
| **文档** | https://alignn.readthedocs.io/ |
| **阅读时间** | 45–60 分钟（全文） |

### 内容概要
**ALIGNN = CGCNN + 线图（Line Graph）。** 核心创新只有一句话：

> CGCNN 的消息传递只在原子之间进行（两体相互作用），但材料的很多性质（如能隙）对**键角**（三体相互作用）非常敏感。ALIGNN 通过额外构建一个"线图"，让键和键之间也能做消息传递，从而**隐式编码了键角信息**。

具体来说：
- **原子图 G**：节点 = 原子，边 = 化学键（同 CGCNN）
- **线图 L(G)**：节点 = 化学键，边 = 共享原子的键对（即键角）
- **ALIGNN 层**：在 G 和 L(G) 之间交替做消息传递

### 性能基准（你需要记住的数字）
| 模型 | 形成能 MAE (meV/atom) | 带隙 MAE (eV) |
|------|----------------------|--------------|
| CGCNN | 44 | 0.25–0.26 |
| SchNet | 37 | 0.21–0.23 |
| MEGNet | 36 | 0.20–0.22 |
| **ALIGNN** | **34** | **0.17–0.19** |

**ALIGNN 相比 CGCNN 在带隙预测上误差降低约 30%。**

### 为什么对 2D 材料特别重要？
- 2D 材料的电子结构（尤其是莫尔超晶格的能隙）对局部几何畸变极其敏感
- 键角变化（如扭转角 θ）直接影响能带折叠和带隙大小
- ALIGNN 通过线图显式建模了这种敏感性 → **理论上比 CGCNN 更适合 2D 材料预测**

### 读的目的
**这是你项目的基石。** 你需要在报告中解释 ALIGNN 的架构、为什么选择它、它的 benchmark 性能如何。这篇论文的 **Fig. 1**（线图构造示意图）和 **Table 1**（性能对比表）是你的报告中最核心的两个图。

### 实操建议
读完论文后，建议直接做：
1. `pip install alignn`
2. 跑 ALIGNN 的预训练模型对几种材料做预测（几分钟就能出结果）
3. 看看代码中 LineGraph 的实现（`alignn/alignn.py`，约 50 行核心代码）

## ⑥ （延伸）DeepH: Deep-learning density functional theory Hamiltonian
**——了解你博士研究的长期方向**

| 项 | 内容 |
|----|------|
| **作者** | Yong Xu, Wenhui Duan group (清华) |
| **来源** | Nature Computational Science 2, 812–823 (2022) |
| **DOI** | 10.1038/s43588-022-00265-6 |
| **阅读时间** | 30 分钟（了解概念即可，不需要深入数学细节） |

### 内容概要
DeepH 不是预测一个标量（如带隙），而是**直接学习整个 DFT 哈密顿量矩阵 $\hat{H}$**。一旦学到了 $\hat{H}$，就可以通过对角化得到**完整的电子结构**（能带、DOS、费米面……一切物理量）。

### 读的目的
不需要在 INFO5000 项目中使用 DeepH，但你需要了解它，因为：
1. **它是你博士研究的长期主线**——双层 hBN 的哈密顿量学习
2. **报告中可以作为"Future Work"**——展示你对领域前沿的理解
3. **帮你理解 ALIGNN 的局限**——ALIGNN 只预测带隙一个数，DeepH 能预测完整电子结构

## 关键概念速查表（物理人专用）

| GNN 术语 | 物理类比 | 在 ALIGNN 中的具体实现 |
|----------|---------|---------------------|
| **Node** | 原子 | 每个原子 = 一个节点，特征向量包含元素属性 |
| **Edge** | 化学键 / 近邻关系 | 截断半径内的原子对，特征 = 相对距离 |
| **Line Graph Edge** | 键角 | 共享一个原子的两个键（三体关联） |
| **Message Passing** | 自洽场迭代（SCF） | 原子间交换"特征信息"，更新自身状态 |
| **Graph Pooling** | 布里渊区积分 | 把所有原子特征聚合为整个晶体的特征 |
| **Readout / FC Layer** | 计算 $\langle \psi \| \hat{O} \| \psi \rangle$ | 全连接层 → 预测带隙 |
| **Loss Function (MSE)** | 最小化能量差 | $\frac{1}{N} \sum (E_g^{\text{pred}} - E_g^{\text{DFT}})^2$ |
| **Training** | 拟合势能面 | 用 JARVIS 数据集优化网络参数 |
| **Overfitting** | 过拟合实验数据 | 用正则化 / 早停 / 交叉验证控制 |
